# Snacks Dictionary (Archived)

Snacks Dictionary was a snack cataloging app where you could rate and track snacks, drinks, and ice creams.

✅ **This project has been merged into the new, unified app:** **배불러! (So Full!)**  
**Live:** https://sofull.site

This repository is now **archived** and kept only for historical reference.

---

## What happened?

Snacks Dictionary and Ramyeon Dictionary were consolidated into a single app that supports multiple categories in one place:
- Ramyeon
- Snacks
- Drinks
- Ice cream

If you were using this app before, the new home for ongoing development and improvements is **배불러! So Full!**.
